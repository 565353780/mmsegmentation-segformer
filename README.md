# MMSegmentation Segformer

## Copy models and datasets

```bash
cd mmsegmentation-segformer
./init.sh
```

## Install MMSegmentation

```bash
conda create -n seg python=3.8 -y
conda activate seg
./setup.sh
```

## Run detect

```bash
python MMSegmentationDetector.py
```

## Run train

```bash
python MMSegmentationTrainer.py
```

## Enjoy it~
